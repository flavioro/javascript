### Ways to Clone Objects in JavaScript (object.assign vs spread operator)
```
const food = { beef: '游꺟', bacon: '游볩' };

Object.assign(food, { beef: '游볼' });
```

console.log(food);
// { beef: '游볼', bacon: '游볩' }

```
const food = { beef: '游꺟', bacon: '游볩' };

const newFood = {
  ...food,
  beef: '游볼',
}
```

console.log(newFood);
// { beef: '游볼', bacon: '游볩' }
